QuizGenius Bookmarklet

Works on ANY device - no extension needed!

Install:

  1. Drag the button below to your bookmarks bar:
QuizGenius

Or manually:

  1. Create a new bookmark
  2. Name it "QuizGenius"
  3. Paste this code in the URL field:

How to use:

  1. Go to your McGraw Hill quiz
  2. Click the QuizGenius bookmark
  3. Enter your API key (first time only)
  4. Click Start

No extension required! Works on school Chromebooks where extensions are blocked.
